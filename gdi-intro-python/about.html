About

This project tracks the slides used for teaching the Intro to Python class with Girl Develop It RDU. The "revisions" listed at the home page represent the state of slides as they appeared during a given set of classes. Each of these revisions is broken into "classes" or "parts" for seperate class sessions or segments of those class sessions.

In general, the most recent set of slides is preferred for anyone seeking to bootstrap their own course with this material or self study. Old revisions are primarily kept available for former students as a reference.
